Консултации с учители

Редовни консултации по всички предмети за подпомагане на учениците при усвояване на учебния материал.

Консултациите се провеждат по график по всички предмети. Учениците могат да посещават консултациите, за да получат допълнителна помощ при разбиране на учебния материал, подготовка за контролни работи или при затруднения в някой предмет.
